The Backend

A description of our server

Overview

Our server runs two Docker containers to host the database and the web server. The dynamic engine of the eyeCU web page is written using Flask. As Flask is single threaded the application is proxied behind NGINX as a web services gateway interface (WSGI). The python application performs all database querying and inserts.

Communication between the IoT devices and the server is achieved over simple HTTP posts. The application accepts data only if it meets a predefined data schema. The sensors send data using this format allowing for a layer of data integrity checking. The next layer of data validation is achieved through statistical analysis.

    About the Project

    The primary goal of the project is to establish a secure network between IoT devices planted throughout the University of Colorado campus. Each device is equipped with various sensors as well as a WiFi interface. Data from each device is sent over to our dedicated server located in the Security Operations Center Lab (SOC-LAB).

    Learn more

    The Red Buffalo Network

    Our network consists of 100 IoT devices carefully placed around our campus as well as three separate docker containers running on our own dedicated server. One docker container holds read only device information and runs this very webserver.
